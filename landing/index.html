Preguntas frecuentes

Lo que preguntan antes de probarlo

¿Mis datos salen de mi empresa?

No. La versión gratuita corre 100% local (web o PC). El plan Enterprise soporta on-premise o nube privada.

¿Necesito saber programar para usarlo?

No. Se instala con un doble clic (portable) o se usa 100% desde el navegador.

¿Puedo migrar desde la herramienta que uso hoy?

Sí, el importador inteligente perfila tu Excel o export actual y sugiere la estructura.

¿El copiloto de IA es obligatorio?

No. El motor de salud, dependencias y priorización funciona 100% sin IA. Claude es una capa opcional que solo redacta mejor lo que el motor ya calculó.

¿Hay soporte en español?

Sí, la plataforma y el soporte son trilingües: español, inglés y portugués.

Empezar ahora

Probalo gratis, hoy

El producto está en fase beta: por ahora se corre localmente en un minuto con Python. El SaaS 100% en la nube es el próximo paso del roadmap.

💻

Local en 1 minuto

Clonar el repositorio y correr:

git clone https://github.com/vieraschiavi/Project-management-mv
./run.sh install && ./run.sh app

📦

Portable para PC, sin instalar

Generá el ZIP portable — doble clic en MV_ProjectManagement.bat y listo:

./run.sh portable

El instalador Windows (.exe) se compila automáticamente en cada release — ver releases.

🧪

Correr los tests

Verificá vos mismo que el motor hace lo que decimos:

./run.sh test

🔌

API para tu BI

Conectá Power BI, Tableau o Excel a la misma data:

./run.sh api
